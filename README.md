# BookListView

## 一、專案簡介

本專案是一個簡易圖書管理程式，主要功能是讓使用者瀏覽圖書資料，並將想借閱的書籍加入借書清單。程式介面左側會顯示多本書籍，包含書籍封面與書名；右側提供「檢視方式」選單，使用者可以切換不同的顯示模式，例如大圖示、詳細資料、小圖示、清單，以及大圖示加詳細資料。下方的借書清單則會顯示使用者已選擇借閱的書籍。

---

## 二、功能說明

### 1. 顯示圖書資料

程式內建 8 本圖書資料，每本書包含：

- 書名
- 作者
- 類別
- 書籍封面圖片

目前收錄的書籍包含：

| 書名 | 作者 | 類別 |
|---|---|---|
| 東方快車謀殺案 | Agatha Christie | 懸疑推理小說 |
| 追風箏的孩子 | Khaled Hosseini | 青春校園小說 |
| 龍紋身的女孩 | Stieg Larsson | 懸疑推理小說 |
| 福爾摩斯探案 | Arthur Conan Doyle | 懸疑推理小說 |
| 沙丘 | Frank Herbert | 科幻小說 |
| 安德的遊戲 | Orson Scott Card | 科幻小說 |
| 別讓我走 | Kazuo Ishiguro | 青春校園小說 |
| 壁花男孩 | Stephen Chbosky | 青春校園小說 |

---

### 2. 切換檢視方式

使用者可以透過右上方的下拉式選單切換圖書顯示方式。

支援的檢視方式包含：

- 大圖示
- 詳細資料
- 小圖示
- 清單
- 大圖示加詳細資料

不同模式可以讓使用者用不同方式瀏覽圖書資料，使介面更有彈性。

---

### 3. 加入借書清單

使用者可以點選想借閱的書籍，程式會跳出確認視窗詢問是否要借閱。

若使用者按下「是」，該書籍會被加入右側的借書清單中。

若該書籍已經存在於借書清單中，程式不會重複加入，避免同一本書被重複借閱。

---

## 三、操作說明

1. 開啟程式後，左側會顯示圖書清單。
2. 使用右上方的「檢視方式」下拉選單，可以切換圖書顯示模式。
3. 點選想借閱的書籍。
4. 程式會跳出確認視窗，詢問是否確定借閱。
5. 按下「是」後，該書籍會加入右側的借書清單。
6. 若同一本書已經加入過借書清單，程式不會重複新增。
